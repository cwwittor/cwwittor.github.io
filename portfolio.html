Work Scheduler

The work scheduler is a scheduler application that saves data to local storage for specific times. The website also has the current date that updates, as well as one-hour time-frames that update with different colors depending on the time of day.

This project utilized the moment API to keep track of time, local storage to save dates, arrays(also 2d arrays) to store data, and logic to determine the time of day with loops.

Weather Dashboard

The weather dashboard displays the current day's weather(temp, humidity, wind speed, UV index) for a searched city along with a 5-day forecast. It also keeps a search history which can be used to search cities.

This project utilized multiple weather API calls to get various information, creation of front end objects in JS, saving values to local storage, using arrays, and usage of jquery.
